# koalas-rpc

#### 项目介绍
考拉RPC

关于名字，为啥叫考拉，原意是《树袋熊》翻译过来的中文名称，
众所周知树袋熊一天睡22个小时，没有最懒只有更懒，
需要本RPC框架可以给所有懒人提供一个可靠，可持续性，可拓展的懒人RPC解决方案
